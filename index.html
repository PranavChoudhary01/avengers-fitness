🚀 Google Apps Script — One-Time Setup (5 min)

Ye ek baar karne ke baad member add/edit/delete karte hi Google Sheet automatically update hogi aur admin ko email jaayega. Kisi bhi device se website khole — data hamesha sync rahega.

1
Google Sheet banao
sheets.new kholao → Spreadsheet ka title rakho "Avengers Fitness". Bas itna — baaki Apps Script khud handle kar lega.
2
Apps Script kholao
Sheet ke andar upar menu mein: Extensions → Apps Script → Nayi tab mein editor khulega.
3
Code paste karo
Saath mein diya avengers-apps-script.js file download karo → pura code copy karo → editor mein existing code hata ke paste karo → Line 8 mein apna Gmail daalo → 💾 Save (Ctrl+S).
⚠️ const ADMIN_EMAIL = "YOUR_EMAIL@gmail.com"; — yahan apna actual email zaroori daalo
4
Deploy karo — Web App
Upar right: Deploy → New Deployment → Type: Web App
• Execute as: Me (apna account)
• Who has access: Anyone
→ Deploy → Google permissions maangega → Allow → Web App URL copy karo
5
URL yahan paste karo → Save
Upar "Google Apps Script URL" wale field mein URL paste karo → Save Settings dabaao.

✅ Done! Ab se member add hote hi — Sheet update + Admin email automatic!
6
⚡ Bonus — Daily Auto Email (Expiry Check)
Apps Script mein baayi taraf ⏰ Triggers icon → Add Trigger
• Function to run: dailyReminderCheck
• Event source: Time-driven → Day timer → 9am to 10am
→ Save → Har roz subah 9 baje automatic expiry check + email aayegi! 🔥
